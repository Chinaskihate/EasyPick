# Infra deployment tutorial
## Preparation
### You have to install:
- #### WSL (if Windows)
- #### Docker
- #### Docker desktop

## Run
- #### docker-compose up

## Applications
- ### RabbitMQ
  - #### UI is running on [http://localhost:15672/](http://localhost:15672/)
  - #### Broker is running on [http://localhost:5672/](http://localhost:5672/)
